# 💊 Pharmacy Medication Reminder

## 📌 Project Description
This project is a Pharmacy Medication Reminder system that helps users remember to take their medicines on time.

## 🚀 Features
- Add medication schedules
- Set reminders
- View daily medications
- Edit or delete reminders

## 🛠️ Technologies Used
- Frontend: React
- Backend: Node.js
- Database: PostgreSQL

## ⚙️ How to Run
1. Download or clone the repository
2. Open the project folder
3. Run the main file
