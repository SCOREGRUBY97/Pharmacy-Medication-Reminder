# Pharmacy Medication Reminder System

## Overview

The Pharmacy Medication Reminder System is a web-based application designed to help users manage their medication schedules effectively. It provides timely reminders, tracks medication adherence, and supports caregiver monitoring to improve overall health outcomes.

This project aims to reduce missed doses and improve medication compliance through a simple and user-friendly platform.

## Features

### User Management

* User registration and secure login
* Password reset functionality
* Role-based access (User, Caregiver, Admin)

### Medication Management

* Add, edit, and delete medications
* Store dosage, frequency, and schedule
* Categorise medications such as morning and evening

### Reminder System

* Automated medication reminders
* Push and SMS notifications
* Repeated alerts for missed confirmations

### Adherence Tracking

* Track taken and missed doses
* View adherence reports
* Monitor medication history

### Caregiver Support

* Link caregiver accounts
* Notify caregivers for missed doses
* Allow monitoring of patient activity

### Dashboard

* View daily medication schedule
* Track upcoming reminders
* Simple and clear user interface

## System Architecture

The system follows a three-tier architecture:

1. Frontend Layer – User interface built with React
2. Backend Layer – API and business logic using Node.js
3. Database Layer – Stores user and medication data
