# Week 10 — Testing

## Expected Deliverable

Perform comprehensive testing to ensure system quality, reliability, and functionality.

---

## Testing Overview

During Week 10, testing activities were conducted to verify that all components of the Pharmacy Medication Reminder system operated correctly. Testing focused on identifying bugs, validating functionality, and ensuring smooth interaction between the frontend, backend, and database components.

Both manual testing and integration testing were performed throughout the system.

---

## Unit Testing

Individual modules were tested independently to verify expected behaviour.

Tested components included:

- Login functionality
- Registration functionality
- Medication management
- Reminder creation
- Caregiver support features
- User profile management

Each module was tested with valid and invalid inputs to ensure proper validation and error handling.

---

## Integration Testing

Integration testing was performed to verify communication between:

- Frontend and Backend
- Backend and Database
- APIs and User Interface

Testing confirmed that information entered through the frontend was successfully stored, retrieved, updated, and displayed correctly.

---

## Critical Test Cases

The following test cases were completed:

- User registration with valid data
- User registration with invalid data
- Successful login
- Failed login attempts
- Add medication
- Edit medication
- Delete medication
- Create reminders
- Reminder status updates
- Caregiver monitoring functions
- Database record verification

---

## Bug Fixes

Issues identified during testing were reviewed and resolved, including:

- Form validation errors
- API response handling
- User interface adjustments
- Data synchronization issues
- Database query corrections

---

## Testing Summary

By the end of Week 10, testing activities confirmed that the system operated according to project requirements. All major features were validated, identified bugs were resolved, and the application was prepared for deployment.
